## :hand: What am I?
Symfony 5 Project to show how to implement and integrate the PayPal's APIs. Including Braintree, V2 Orders, Payments, Billing, Connect with PayPal etc...

## :floppy_disk: How we get working?
```php
Composer install
yarn install
docker-compose up -d
```
